    • What is Möbius Modeller?

    Möbius Modeller is an open-source web application for 3D parametric modelling. The app allows users to create modelling procedures using the Öbi visual programming language. In the Öbi language, you write computer programs by drawing flowcharts and by defining procedures using 'click and fill-in-the-blanks' coding.

    Since Möbius Modeller is a web app, you can just go ahead an use it.

    • The software is free, so there is no need to pay!
    • The software is online, so there is no need to install anything on your computer.
    • The software does not require authentication, so there is no need to create an account or to log in.
    • Code is executed in your browser, so there is no need to worry about network speed.
    • Your data never leaves your browser, so there is no need to worry about data privacy and security.

    The Möbius models that you create can be published as interactive web pages, allowing visitors on the web to explore the models.

    Getting Started

    The best way to start using Möbius Modeller is to explore some of the models in the gallery. After that you can dive in and start creating your own flowcharts and procedures.

    The icons at the top of the page allow you to navigate into your model, from the top level dashboard down the procedures at the bottom level.

    Gallery The gallery contains a selection of example models for you to explore. To access the gallery, click view_comfy
    Dashboard Each model has its own dashboard that exposes a set of parameters, that can be set using sliders and UI components. To access the dashboard, click
    Flowchart Each model is has an Öbi flowchart that defines a set of nodes connected with edges. To access the flowchart, click
    Procedure Each node in the model has its own procedure, defined using the Öbi language. To access the procedure, click
    Execute To execute the Öbi flowchart, click the execute button. This will transpile your Öbi code into Javascript an execute it in your browser. play_circle_outline
    View To view the results, look at the model in the 3D viewer on the right. Also, you can check the console in case there were any errors.
